🔍 Compare Your Menu

This tool overlays your CSS-generated menu on top of the original KG website screenshot so you can check for pixel-perfect accuracy.

  1. Enter your GitHub Pages URL — e.g. https://username.github.io/personal-page/hw8/
  2. Enter the KG screenshot URL — from the assignment email you received
  3. Set the width — the max-width value from your assignment email
  4. Click Load to compare

💡 Tip: Check the 🔴 Red Color box to make your CSS text red — this makes it easy to spot differences between your text and the original white menu text underneath.
