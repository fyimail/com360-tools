🔍 Compare Any Grid Section

This tool overlays your CSS page on top of the original KG website screenshot so you can check for pixel-perfect accuracy — for the menu, hero, or any grid box.

  1. Enter your GitHub Pages URL — e.g. https://username.github.io/personal-page/hw8/
  2. Enter the KG screenshot URL — from the assignment instructions
  3. Set the width — the max-width value from your assignment
  4. Click Load to compare
  5. Use the 📦 Section bar to switch between grid boxes (Menu, Hero, Box 3, etc.) or set custom Top & Height values

💡 Tip: Check the 🔴 Red Color box to make your CSS text red — this makes it easy to spot differences between your text and the original text underneath.
